### 짝수와 홀수

###### 문제 설명

정수 num이 짝수일 경우 Even을 반환하고 홀수인 경우 Odd를 반환하는 함수, solution을 완성해주세요.

##### 제한 조건

- num은 int 범위의 정수입니다.
- 0은 짝수입니다.



#### Java

~~~java
class Solution {
    public String solution(int num) {
      	//나머지 값으로 홀수, 짝수 비교
        if(num%2 == 0){
            return "Even";
        }else{
            return "Odd";
        }
    }
}
~~~



#### Javascript

~~~javascript
function solution(num) {
  	//나머지 값으로 홀수, 짝수 비교
    if(num%2 === 0){
        return "Even";
    }else{
        return "Odd";
    }
}
~~~
